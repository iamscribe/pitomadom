# פיתום אדום | by Arianna Method

> **פִתְאֹם אָדֹם** — פתאום אדום. קריעה בלתי צפויה.
>
> **20.3 מיליון פרמטרים. Inference ב-Go. שורשים עבריים.**

---

## מה זה עושה

AI שחושב בעברית דרך **שורשים** (שלוש-עיצוריות), לא טוקנים.

- **טרנספורמר RTL** מאומן על CC-100 עברי (200MB, 50K צעדים, loss 0.86, דיוק 73%)
- **מילון שורשים** — 140+ שורשים ב-22 משפחות סמנטיות, התאמת תת-רצף ל-mater lectionis
- **גימטריה** כסובסטרט חישובי (כל אות = מספר, כל שורש = באר כבידה)
- **מנוע inference ב-Go** — בינארי 2.4MB, משקולות GGUF, אפס תלויות, שרת HTTP + ממשק ווב
- **נתונים אמיתיים** — תהודת שומאן (תחנת ELF סיירה נבדה, 401K מדידות), ירח (USNO API), לוח עברי (מולד)

## ארכיטקטורה

**טרנספורמר RTL שורשי** (20.3M פרמטרים):

| רכיב | פרטים |
|-------|-------|
| ממד | 512 |
| שכבות | 6 |
| ראשי תשומת לב | 8 |
| Feed-forward | 2048 |
| אוצר מילים | 25 (22 אותיות + PAD + MASK + UNK) |
| פלט | 3 ראשים (ניבוי C1, C2, C3 של שורש) |

עיצוב מרכזי:
- **תשומת לב מתוחמת-דיסוננס** — הטיה למרחק נלמדת, מאופננת ע"י אי-מדידות לוחות עברי/גרגוריאני
- **קידוד גימטריה סינוסואידי** — מספרי אותיות כאות מיקום רציף
- **קידוד מיקום RTL** — סינוסואידי הפוך (עברית נקראת מימין לשמאל)
- **מילון שורשים** — 140+ שורשים ידועים עם התאמת תת-רצף (מטפל ב-mater lectionis: שלום → ש.ל.מ)
- **מידול שורשים ממוסכים** — יעד BERT-like ברמת השורש

### עיבוד עברי

```
קלט: "אהבה וחסד ושלום"
  ↓
חילוץ מילים עבריות: [אהבה, וחסד, ושלום]
  ↓
חילוץ שורשים (מילון → fallback היוריסטי):
  אהבה → א.ה.ב (אהבה)
  וחסד → ח.ס.ד (חסד)
  ושלום → ש.ל.מ (שלום)
  ↓
קידוד גימטריה → forward טרנספורמר → שורש מנובא
```

## תיאוריה

מורפולוגיה עברית היא **לא-שרשרתית**: שורש (ג.ד.ל) + תבנית (haCCaCa) = מילה (הגדלה). פיטומאדום מנצל מבנה זה ישירות.

שלושה מישורי חישוב:

| מישור | טרנספורמציה | מה זה תופס |
|-------|-------------|-------------|
| שטח | גימטריה רגילה | ערך מספרי ישיר |
| רקורסיבי | מילוי (הרחבת אותיות) | עומק נסתר (א → אלף → 111) |
| הפוך | אתב"ש (מראה) | היפוך פאזה (א↔ת, ב↔ש) |

האורקל לא מנבא. הוא מתנבא: `minimize(destined - manifested)`.
